1. Introduction to Python

What is Python?

  • Python is a high-level, interpreted programming language
  • Created by Guido van Rossum in 1991
  • Known for simple and readable syntax
  • Supports multiple programming paradigms (procedural, OOP, functional)

Advantages of Python:

  • Easy to learn and read
  • Large standard library
  • Interpreted language (no compilation needed)
  • Platform independent
  • Supports dynamic typing

2. Python Setup and Installation

Installation Steps:

  1. Download Python from www.python.org
  2. Run the installer
  3. Add Python to PATH (important)
  4. Verify installation: python --version

IDEs and Editors:

  • IDLE (included with Python)
  • PyCharm
  • VS Code
  • Jupyter Notebook

3. Variables and Data Types

Variables:

name = "Python"      # String
age = 30             # Integer
height = 5.9         # Float
is_active = True     # Boolean

Data Types:

  • int: Integer numbers (-5, 0, 100)
  • float: Decimal numbers (3.14, 2.5)
  • str: Text (strings) ("Hello", 'World')
  • bool: Boolean (True, False)
  • list: Ordered collection [1, 2, 3]
  • dict: Key-value pairs {"name": "John"}
  • tuple: Immutable collection (1, 2, 3)
  • set: Unique items {1, 2, 3}
# Checking data type
print(type(5))      # 
print(type(3.14))   # 
print(type("text")) #

4. Operators

Arithmetic Operators:

Operator Description Example
+ Addition 10 + 5 = 15
- Subtraction 10 - 5 = 5
* Multiplication 10 * 5 = 50
/ Division 10 / 5 = 2.0
// Floor Division 10 // 3 = 3
% Modulus 10 % 3 = 1
** Exponentiation 2 ** 3 = 8

Comparison Operators:

== (equal to)
!= (not equal to)
> (greater than)
< (less than)
>= (greater than or equal to)
<= (less than or equal to)

Logical Operators:

and  # Both conditions true
or   # At least one condition true
not  # Negates the condition

Assignment Operators:

= (assign)
+= (add and assign)
-= (subtract and assign)
*= (multiply and assign)
/= (divide and assign)

5. Input/Output Operations

Output with print():

print("Hello, World!")
print("Name:", "John", "Age:", 25)
print(10 + 5)
x = 50
print(f"The value is {x}")

Input with input():

name = input("Enter your name: ")
age = input("Enter your age: ")
print("Name:", name, "Age:", age)

6. Type Conversion

# Convert to integer
num = int("123")      # 123
num = int(45.67)      # 45

# Convert to float
num = float("3.14")   # 3.14
num = float(10)       # 10.0

# Convert to string
text = str(123)       # "123"
text = str(3.14)      # "3.14"

# Convert to boolean
bool_val = bool(1)    # True
bool_val = bool(0)    # False
