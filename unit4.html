Unit 4: Data Structures

Topics Covered:

  • Lists
  • Tuples
  • Dictionaries
  • Sets
  • Strings
  • File Handling

1. Lists

Lists are ordered, mutable collections of items.

fruits = ["apple", "banana", "cherry"]
numbers = [1, 2, 3, 4, 5]
mixed = [1, "hello", 3.14, True]

# Accessing elements
print(fruits[0])      # apple
print(fruits[-1])     # cherry

# Slicing
print(fruits[0:2])    # ['apple', 'banana']

List methods:

list = [3, 1, 4, 1, 5]

list.append(9)        # Add element
list.extend([2, 6])   # Add multiple elements
list.insert(0, 7)     # Insert at position
list.remove(1)        # Remove first occurrence
list.pop()            # Remove last element
list.sort()           # Sort in place
list.reverse()        # Reverse order
list.clear()          # Remove all elements

print(list.index(4))  # Find index
print(list.count(1))  # Count occurrences

List comprehension:

squares = [x**2 for x in range(5)]      # [0, 1, 4, 9, 16]
evens = [x for x in range(10) if x % 2 == 0]  # [0, 2, 4, 6, 8]

2. Tuples

Tuples are ordered, immutable collections.

colors = ("red", "green", "blue")
single = (1,)         # Note the comma for single element
mixed = (1, "hello", 3.14)

# Accessing elements
print(colors[0])      # red
print(colors[1:])     # ('green', 'blue')

# Unpacking
r, g, b = colors
print(r, g, b)

Tuple operations:

t = (1, 2, 3, 2)
print(len(t))         # 3
print(t.count(2))     # 2
print(t.index(3))     # 2

# Concatenation
t1 = (1, 2)
t2 = (3, 4)
t3 = t1 + t2          # (1, 2, 3, 4)

# Repetition
t = (1, 2) * 3        # (1, 2, 1, 2, 1, 2)

Why use tuples?

  • Immutability ensures data integrity
  • Can be used as dictionary keys
  • Slightly faster than lists
  • Prevent accidental modifications

3. Dictionaries

Dictionaries are unordered, mutable collections of key-value pairs.

student = {
    "name": "John",
    "age": 20,
    "city": "NYC"
}

# Accessing values
print(student["name"])       # John
print(student.get("age"))    # 20

# Adding/Modifying
student["age"] = 21
student["email"] = "john@example.com"

# Removing
del student["email"]
student.pop("city")

Dictionary methods:

d = {"a": 1, "b": 2, "c": 3}

d.keys()              # Get all keys
d.values()            # Get all values
d.items()             # Get key-value pairs
d.clear()             # Remove all items
d.copy()              # Create a copy
d.update({"d": 4})    # Update with new items

print("a" in d)       # Check if key exists
print(d.get("a", 0))  # Get with default value

Dictionary comprehension:

squares = {x: x**2 for x in range(5)}  # {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}
words = ["apple", "banana", "cherry"]
word_lengths = {w: len(w) for w in words}

4. Sets

Sets are unordered collections of unique items.

colors = {"red", "green", "blue"}
numbers = {1, 2, 3, 2, 1}  # {1, 2, 3} - duplicates removed

# Adding elements
colors.add("yellow")

# Removing elements
colors.remove("red")      # Raises error if not found
colors.discard("green")   # No error if not found

# Set operations
a = {1, 2, 3}
b = {3, 4, 5}

union = a | b             # {1, 2, 3, 4, 5}
intersection = a & b      # {3}
difference = a - b        # {1, 2}
symmetric = a ^ b         # {1, 2, 4, 5}

Set methods:

s = {1, 2, 3}
s.clear()                 # Remove all
s.copy()                  # Create copy
s.union({4, 5})           # {1, 2, 3, 4, 5}
s.intersection({2, 3, 4}) # {2, 3}
s.difference({2})         # {1, 3}

5. Strings

Strings are sequences of characters (immutable).

text = "Hello, World!"

# Accessing characters
print(text[0])            # H
print(text[-1])           # !
print(text[0:5])          # Hello

# String methods
print(text.lower())       # hello, world!
print(text.upper())       # HELLO, WORLD!
print(text.replace("World", "Python"))
print(text.split(", "))   # ['Hello', 'World!']
print("  hello  ".strip())  # hello

# String formatting
name = "Alice"
age = 25
print(f"{name} is {age} years old")
print("{} is {} years old".format(name, age))

String methods:

s = "hello world"
s.find("world")           # 6
s.count("l")              # 3
s.startswith("hello")     # True
s.endswith("world")       # True
s.isdigit()               # False
s.isalpha()               # False
s.replace("world", "python")
s.capitalize()            # "Hello world"
s.title()                 # "Hello World"

6. File Handling

Reading files:

with open("file.txt", "r") as f:
    content = f.read()    # Read entire file
    
with open("file.txt", "r") as f:
    lines = f.readlines() # Read as list of lines
    
with open("file.txt", "r") as f:
    for line in f:
        print(line.strip())

Writing files:

with open("file.txt", "w") as f:
    f.write("Hello, World!")

with open("file.txt", "a") as f:
    f.write("\nNew line")  # Append

File modes:

  • r: Read (default)
  • w: Write (overwrites)
  • a: Append
  • x: Create new file
  • b: Binary mode

Handling exceptions:

try:
    with open("file.txt", "r") as f:
        content = f.read()
except FileNotFoundError:
    print("File not found")
except IOError:
    print("Error reading file")
